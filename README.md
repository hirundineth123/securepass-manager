# 🔐 SecurePass Manager

A modern, locally encrypted password manager built with Python and PySide6.

## 🚀 Features

- 🔐 Master password protection
- 🧠 AES-based encryption using `cryptography`
- 💾 Local encrypted vault storage
- 🖥️ Simple GUI interface
- ➕ Add and manage credentials

## 🛠️ Tech Stack

- Python
- PySide6 (GUI)
- Cryptography (Fernet / AES)
- JSON-based storage (encrypted)
